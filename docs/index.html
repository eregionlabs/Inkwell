Frequently Asked Questions

How does the free trial work?

InkBlot is free to download and use for 3 days with full functionality. After the trial, a one-time $0.99 in-app purchase unlocks the app forever — no subscription.

How do I restore my purchase on a new Mac?

Open InkBlot and click "Restore Purchase" on the unlock screen. Make sure you're signed into the same Apple ID you used for the original purchase.

What markdown features are supported?

InkBlot supports standard Markdown: headings, bold, italic, links, images, code blocks, blockquotes, tables, lists, and horizontal rules.

Why does my PDF look exactly like the preview?

Both the live preview and PDF export use the same rendering engine and stylesheet. What you see in the editor is what you get in the PDF — no formatting surprises.

What macOS versions are supported?

InkBlot requires macOS 12 (Monterey) or later.

Does InkBlot require an internet connection?

No. InkBlot works entirely offline. The only network access is for verifying your App Store purchase, handled by macOS.

Troubleshooting

PDF export produces a blank file

Make sure you have content in the editor before exporting. If the issue persists, try saving the file first (Cmd+S), then export again.

Purchase not recognized after reinstall

Click "Restore Purchase" on the unlock screen. If that doesn't work, make sure you're signed into the correct Apple ID in System Settings > Apple ID. Then relaunch InkBlot.

App won't open or crashes on launch

Try deleting and reinstalling from the Mac App Store. If the problem continues, email us with your macOS version and we'll investigate.

File won't open or save

InkBlot can open and save .md and .txt files. If a file won't open, check that it's a plain text file and not a binary format.

Keyboard Shortcuts

  • Cmd+O — Open file
  • Cmd+S — Save file
  • Cmd+E — Export to PDF

Privacy Policy

InkBlot does not collect, store, or transmit any personal data. Your files remain on your device at all times. We do not use analytics, tracking, cookies, or third-party services. The app requires no account and no internet connection to function.

The only network activity is the standard App Store purchase verification handled by macOS, which is governed by Apple's Privacy Policy.
