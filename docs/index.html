InkBlot

Markdown editor with pixel-perfect PDF export

About

InkBlot is a minimal markdown editor for macOS. Write on the left, see a beautiful live preview on the right, and export PDFs that look exactly like your preview.

Both the preview and PDF export use the same rendering engine and stylesheet, so what you see is what you get — no formatting surprises.

Features

  • Live split-pane editor with instant preview
  • PDF export that matches your preview exactly
  • Clean typography optimized for comfortable reading
  • Open, edit, and save any .md file
  • Resizable panes and keyboard shortcuts
  • 3-day free trial, then $0.99 to unlock forever

Privacy Policy

InkBlot does not collect, store, or transmit any personal data. Your files stay on your device. We do not use analytics, tracking, or third-party services. The app requires no account and no internet connection to function.
